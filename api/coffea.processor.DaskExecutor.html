Execute using dask futures

Parameters:

  • items (list) ‚Äì List of input arguments

  • function (callable) ‚Äì A function to be called on each input, which returns an accumulator instance

  • accumulator (Accumulatable) ‚Äì An accumulator to collect the output of the function

  • client (distributed.client.Client) ‚Äì A dask distributed client instance

  • treereduction (int, optional) ‚Äì Tree reduction factor for output accumulators (default: 20)

  • status (bool, optional) ‚Äì If true (default), enable progress bar

  • compression (int, optional) ‚Äì Compress accumulator outputs in flight with LZ4, at level specified (default 1) Set to None for no compression.

  • priority (int, optional) ‚Äì Task priority, default 0

  • retries (int, optional) ‚Äì Number of retries for failed tasks (default: 3)

  • heavy_input (serializable, optional) ‚Äì Any value placed here will be broadcast to workers and joined to input items in a tuple (item, heavy_input) that is passed to function.

  • function_name (str, optional) ‚Äì Name of the function being passed

  • use_dataframes (bool, optional) ‚Äì

    Retrieve output as a distributed Dask DataFrame (default: False). The outputs of individual tasks must be Pandas DataFrames.

    Note

    If heavy_input is set, function is assumed to be pure.
